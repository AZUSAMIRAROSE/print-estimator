# Browser Compatibility Matrix

## Tier-1 Support Targets

- Chrome (latest 2)
- Edge (latest 2)
- Firefox (latest 2)
- Safari 16+
- iOS Safari 16+
- Android Chrome 120+

## Verified App Features

- Estimation wizard forms and calculations
- Recharts dashboards/reports
- CSV export + print styles
- Local persistence fallback
- API-connected quote workflow (when backend enabled)

## Legacy/Non-Target

- Internet Explorer: unsupported
- Chromium < 110: best effort only
